# Repositorio de Despliegue de Aplicaciones Web (DDAW)

Este repositorio pertenece al módulo **Despliegue de Aplicaciones Web (DDAW)** del 2º curso del ciclo de **Desarrollo de Aplicaciones Web (DAW)**.  
Aquí se guardan las diferentes prácticas, ejercicios y proyectos que se realizan a lo largo del módulo.

## 🌳 Estructura del repositorio

### 🐙 **GitHub : Git**
👉 Apuntes y prácticas relacionadas con el uso de **Git**: comandos básicos, flujo de trabajo, operaciones de commit, push, pull, etc.

### 🐙 **GitHub : Git y ramas**
👉 Práctica de trabajo con **ramas** y repositorios.  
📌 Ejemplo de referencia: https://github.com/GsuDev/DAW-Practica-Grupal-1

### 🐙 **GitHub : Git : proyecto_software**
👉 Material del bloque de Git aplicado a un proyecto completo.  
Contiene organización, prácticas, ejemplos de flujo real y ficheros del proyecto dentro de `GitHub/Git/proyecto_software`.

### 🐋 **Docker : Ejercicio Docker + precommits + flask**
👉 Prácticas de contenedorización y ejecución de aplicaciones con **Docker**, incluyendo:
- Configuración de **pre-commits**
- Ejemplo de aplicación con **Flask**

### 🪣 **SonarQube : Formateo, Estilo de Código y Análisis**
👉 Configuración de **SonarQube** para análisis estático, calidad de código y buenas prácticas.

### 🦝 **Apache**
👉 Contiene ejercicios y apuntes sobre el servidor web **Apache**:
- Configuración
- Despliegue de proyectos
- Módulos y ejemplos prácticos

### 🧭 **DNS**
👉 Ejercicios y documentación sobre el sistema de nombres de dominio:
- Resolución
- Configuraciones
- Conceptos fundamentales

## 🎯 Objetivo del módulo

📌 El objetivo de este módulo es aprender a preparar, desplegar y mantener aplicaciones web en entornos reales y profesionales, empleando herramientas modernas y buenas prácticas de la industria.
